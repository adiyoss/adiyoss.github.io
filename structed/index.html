LaStrada - Java package for Structured Problems

Open/Close Menu


La Strada is a collection of machine learning algorithms for structured prediction problems. In structured prediction each task is distinctive and has its own set of feature functions, unique measure of performance and in many cases a non-standard inference. La Strada package was designed to handle this inherent implementation complexity by having an easy interface to user-defined feature functions, decoder and evaluation function
